第一步：使用font-face声明字体
@font-face {
    font-family: "yxl-font";
    src: url("yxl-font.eot");
    src: url("yxl-font.eot?#iefix") format("embedded-opentype"),
    url("yxl-font.woff") format("woff"),
    url("yxl-font.ttf") format("truetype"),
    url("yxl-font.svg#yxl-font") format("svg");
}
第二步：定义使用yxl-font的样式
.yxl-font {
    font-family:"yxl-font" !important;
    font-size:16px;
    font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;
}
第三步：挑选相应图标并获取字体编码，应用于页面
<i class="yxl-font">&#x33;</i>
